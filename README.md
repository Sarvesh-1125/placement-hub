# Placement Hub

This project is a simple Placement Portal Application built using Flask and SQLite.

## Features

* User registration and login
* Role-based access (Admin, Student, Company)
* Company can add job postings
* Students can view and apply for jobs

## Technologies Used

* Python (Flask)
* SQLite Database
* HTML and CSS

## How to Run

1. Install Flask
2. Run main.py
3. Open browser at http://127.0.0.1:5000

## Description

This system allows interaction between students and companies. Companies can post job opportunities and students can apply for them. The project demonstrates basic web development concepts such as routing, database integration, and role-based authentication.
